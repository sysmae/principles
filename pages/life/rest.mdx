# 휴식에 관한 원칙

### 1.공허한 휴식

정의 : 100%가 아닌 휴식

현황 : 10대,20대 / 30,40대


10대,20대 : 휴식에 대한 터부시, 죄책감\
휴식 자체를 완전히 경험해야겠다는 생각을 못함\
쉬는 것에 죄책감이 들고 항상 바빠야한다는 강박이 있다.


30대,40대 : 휴식해야한다고 느껴도 100% 제대로 쉬었다는 느낌을 못 느낌\
쉬긴 쉬어도 100% 제대로 잘 쉬었다는 느낌이 들지 않음

전환 : 충만한 휴식\
대안 : 치열 + 본질

### 2.치열한 휴식

정의 : 휴식에 모든 오감/능력에 집중, 몰입, 100% 경험\
능동적 휴식, 밀리고 밀리다가 어쩔 수 없이 휴식하는 수동적 휴식과는 다르다.

행동1 : 기본 조건 조성\
(휴식 방해/반대 요소 차단 + 온전한 휴식 시간/환경 확보 by Skill)

행동2 : 휴식 자체 몰입\
(100% 전체를 경험하는데 오감/능력 집중 by본질)

### 3.충만한 휴식

정의 :\
(이론적) 치열 + 본질 = 100% 휴식 경험\
(체감적)'잘 쉬었다' = 뇌/감정 재충전 = 원상태 복귀

기준 : '충만한 휴식' 경험할 때까지\
Why. 삶의 변화 | 쾌,불쾌 발견 | 충만한 인생\
How1. 재시청->재시도->재시청->재시도...\
How2. 커뮤니티 도움 = 이성적 + 감성적
